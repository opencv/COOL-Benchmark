OpenCV Benchmark Code

This is the actual Python code executed on each EC2 instance to benchmark OpenCV performance:

# Process images with OpenCV operations
for iteration in range(iterations):
    for img_b64 in images_b64[:20]:  # Process first 20 images
        # 1. Decode image from base64
        img_data = base64.b64decode(img_b64)
        img_array = np.frombuffer(img_data, dtype=np.uint8)
        img = cv2.imdecode(img_array, cv2.IMREAD_COLOR)
        
        # 2. Resize maintaining aspect ratio to 512xY
        h, w = img.shape[:2]
        if w > h:
            new_w, new_h = 512, int(512 * h / w)
        else:
            new_w, new_h = int(512 * w / h), 512
        img_resized = cv2.resize(img, (new_w, new_h), interpolation=cv2.INTER_LINEAR)
        
        # 3. Convert to grayscale for contour detection
        gray = cv2.cvtColor(img_resized, cv2.COLOR_BGR2GRAY)
        
        # 4. Apply Gaussian blur to reduce noise
        blurred = cv2.GaussianBlur(gray, (5, 5), 0)
        
        # 5. Apply Otsu's thresholding
        _, thresh = cv2.threshold(blurred, 0, 255, cv2.THRESH_BINARY + cv2.THRESH_OTSU)
        
        # 6. Find contours
        contours, _ = cv2.findContours(thresh, cv2.RETR_EXTERNAL, cv2.CHAIN_APPROX_SIMPLE)
        
        # 7. Draw contours in green
        cv2.drawContours(img_resized, contours, -1, (0, 255, 0), 2)
        
        # 8. Encode back to JPEG (in memory, no filesystem access)
        _, buffer = cv2.imencode('.jpg', img_resized, [cv2.IMWRITE_JPEG_QUALITY, 85])
        processed_b64 = base64.b64encode(buffer).decode('utf-8')

Operations Benchmarked:

  • decode - Base64 decode and cv2.imdecode()
  • resize - cv2.resize() with INTER_LINEAR interpolation
  • cvtColor - BGR to grayscale conversion
  • blur - cv2.GaussianBlur() with 5x5 kernel
  • threshold - cv2.threshold() with Otsu's method
  • findContours - cv2.findContours() with RETR_EXTERNAL
  • drawContours - cv2.drawContours() with 2px thickness
  • encode - cv2.imencode() to JPEG with quality 85

Note: All benchmarks use identical code and parameters to ensure fair comparison. The only difference is the OpenCV build (COOL marketplace AMI vs DIY from source) and instance type.
